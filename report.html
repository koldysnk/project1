Coherence and Contrast

  • Are the images appropriate to the theme?
    Yes, all images were provided by Grace Barnes and are pictures of her work.
  • Are the images appropriately sized?
    Yes, they are all sized so that their proportions are not distorted and they do not take up too much space. They are also big enough so you can see the details.
  • Are there strong axes of alignment?
    Yes, accross the entire site I maintained a left axes of alignment with three exceptions. First the title is center aligned. Second the nav bar is right aligned. Third the instagram link is centered.
  • Is your style maintained across pages?
    Yes, I use the same color theme and font theme accrose the entire site. Pages with common styles share common style sheets to ensure this.
  • Is the spacing clean?
    Yes, items that are related are closer than items that are not. The size of the spacing also changes based on how close the items are related.
  • Does the site include different fonts?
    Yes, for the body of the text, I use the google font Roboto with the sans-serif style. For h1 texts, I use the Roboto-slab font with the serif style. For the title in the landing page and the h2 headers, I use Roboto condensed with the sans-serif style.

Client-mindedness

  • Is the site designed with the client in mind rather than the designer?
    Yes, Grace helped decide what the five pages were going to be and approved layouts I provided. Grace also helped decide the color scheme.
  • Does your site appear appropriately on both mobile and desktop screen?
    Yes, I used media queries to define layouts for three different screen sizes: screens with a width less than 650, less than 1200, and above 1200. When the screen is less than 650 the nav bar changes to a mobile version.
  • Is your site arranged in a navigable way?
    Yes, there is both a header with nav links and a footer with nav links. Both of these adjust with the size of the device. Both of these are available on every page. Section titles also contain nav links.
  • Are images made accessible with alt attributes?
    Yes, every image has a descritption in an alt tag.

Project Organization

  • Are external CSS files used?
    Yes, the main.css file contains styles that are used by all HTML pages. The artwork.css file contains styles that are used by pages that contain an artwork card. The index, about, and contact CSS pages only contain styles relevant to those pages. The report.css file contains styles for this page.
  • Is the site organized into files as specified?
    Yes, all HTML files are at the root level (in the project1 folder). All CSS files are in the css folder. All JavaScript files are in the js folder. All images are in the images folder.
  • Are classes and IDs used appropriately to apply styles at the right level of granularity?
    Yes, IDs are only used once per ID and classes are only used for items of the same tag that have the same style. ID and class names are also descriptors of what the item represents in the site.
  • Are selectors used appropriately?
    Yes, any time a style is applied the lowest level of selectors are used. A class name or ID is used if avalable or the '>' key is used to select children of a certain type.
